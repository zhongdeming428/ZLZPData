# 简介
这个仓库的GitHub Pages用于展示我使用Node爬虫在智联招聘网站上爬取的数据。

数据囊括全国27个主要城市的7个主要软件岗位，共爬取数据27000余条（截止到2018年2月8号），虽然数据具有片面性，可靠性不是很高，但是也具有一定的代表性。具体有几分可靠性，各位自行判断，作者不承担任何后果。
